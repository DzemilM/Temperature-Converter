# Temperature Converter

A small vanilla-JavaScript app to practice **DOM manipulation**, **form handling**, and **basic calculations**.

Enter a temperature, pick the unit to convert **from** and **to**, and hit **Convert**. The button stays disabled until all three fields are filled.

## Features

- Convert between **Celsius**, **Fahrenheit**, and **Kelvin**
- Convert button is enabled only when all three fields are filled in
- Result displayed below the form

## Run it

No build step, no dependencies — it's plain HTML/CSS/JS.

1. Clone or download this folder.
2. Open `index.html` in any browser (double-click it, or drag it into a tab).
3. Edit `script.js` and refresh the page to see changes.

## Project structure

| File         | Purpose                                    |
| ------------ | ------------------------------------------ |
| `index.html` | Page structure (input, dropdowns, button)  |
| `styles.css` | Styling                                    |
| `script.js`  | Button-enable logic and conversion math    |
